# 🌌 SpaceScope

**Explore the Cosmos In Real-Time.** 
Advanced telemetry, 3D orbital tracking, and mission control for the modern space age. Built for **Code-A-Thon 2.0** at Datta Meghe College of Engineering (DMCE), Navi Mumbai.

---

## 🚀 Overview
SpaceScope is a comprehensive space monitoring dashboard designed to bring advanced telemetry and cosmic data into a sleek, accessible interface. Whether you are tracking active satellites in Earth's orbit, monitoring space weather, or following the latest ISRO and SpaceX missions, SpaceScope provides a unified Mission Control experience.

## ✨ Key Features

### 🌍 Real-Time 3D Tracking
- **WebGL Globe Rendering:** Live, interactive 3D visualization of Earth, tracking over 2,800 active satellites and orbital debris.
- **Satellite Tracker:** Expandable tracking modules to view specific trajectories and satellite telemetry.

### 🛰️ Mission Registry & Control
- **Live Mission Updates:** Track ongoing and upcoming missions from major space agencies (Artemis, SpaceX, ISRO).
- **Cosmic Weather Report:** Real-time data on Solar Wind velocity, Geomagnetic indices (e.g., G2 Moderate), Solar Flare risks, and Sunspot counts.

### 🛡️ Earth Impact Monitoring
- **24/7 Observation:** Utilize satellite data to monitor critical Earth metrics, including agricultural health, pollution levels, and disaster response.
- **Active Alerts:** Get notified about active global or cosmic anomalies.

### 🔭 Educational & Planning Tools
- Integrated tools for learning about space, tracking Moon Phases, and an Event Calendar for upcoming launches and astronomical events.

## 🛠️ Tech Stack (Suggested)
- **Frontend:** React.js
- **Styling:** Tailwind CSS
- **3D Rendering:** WebGL, Three.js, react-globe.gl
- **Data & APIs:** NASA Open APIs, The Space Devs API, and other telemetry providers.
